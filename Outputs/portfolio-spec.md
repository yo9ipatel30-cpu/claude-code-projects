## Design Specification

### Color Palette

| Token | Hex | Use |
|-------|-----|-----|
| Navy (primary) | `#1B2A4A` | Nav, headings, section titles |
| Slate blue (accent) | `#4A7FA5` | Links, hover states, tags, CTAs |
| Off-white (background) | `#F7F8FA` | Page background |
| White | `#FFFFFF` | Cards, nav bar |
| Text | `#1A1A1A` | Body copy |
| Muted text | `#6B7280` | Dates, subtitles, metadata |

### Typography

| Use | Font | Weight |
|-----|------|--------|
| Headings | Inter | 700 |
| Body | Inter | 400 |
| Labels / tags | Inter | 500 |

Loaded from Google Fonts. No custom font files needed.

### Layout & Behavior

- Fixed top nav bar: name logo (left) + page links (right)
- Pages: Home · About · Research · Writing · Leadership & Service · Contact
- Smooth page transitions (CSS fade-in on load, single pass — no looping)
- Research positions and publications displayed as cards with hover shadow
- Mobile-responsive: hamburger menu on small screens, stacked layouts
- Light mode only — no toggle (keeps design consistent and simple)

### Hosting

YogiPatel.com pointed to GitHub Pages — free hosting, static files, no build step. Each page is a separate HTML file sharing one CSS file and one JS file. Structure:
